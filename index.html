    Best Practices

    • Be specific about context, tone, and output format
    • Use few-shot examples when possible
    • Test variations and refine iteratively
    • Include safety guardrails for user-facing responses

    Trend and Market Research

    Research Resources

    • Google Trends – track keyword demand
    • Product Hunt – see trending tools
    • Reddit communities like r/InternetIsBeautiful, r/ChromeExtensions

    Performance Tracking

    • Chrome Web Store Developer Dashboard (installs, ratings)
    • Google Analytics 4 for user behavior
    • Sentiment analysis on reviews

    Production Workflow

    Development Process

    • Create wireframes using Figma or Excalidraw
    • Use Git + GitHub for version control
    • Adopt CI/CD pipelines (GitHub Actions, etc.)

    Testing and QA

    • Write unit tests with Jest or Puppeteer
    • Conduct manual usability testing
    • Run security audits (CSP, permissions minimization)

    User Persona Development

    • Create empathy maps for your target users
    • Map out user journey from install to action
    • Validate assumptions through interviews

    Launch & Monetization Strategy

    Launch Tactics

    • Build email list pre-launch with landing page
    • Submit to Product Hunt on launch day
    • Leverage Twitter/X, LinkedIn, Indie Hackers

    Monetization Models

    • Freemium: Free base + paid premium features
    • Subscription: Monthly access (Stripe/BillingJS)
    • Affiliate: Recommend tools/products
    • Ads: Use AdSense or direct sponsorships

    Data-Driven Decisions

    • Set up A/B testing for CTAs and pricing
    • Analyze funnel drop-offs
    • Optimize based on retention and conversion

    Post-Launch Optimization

    User Feedback Collection

    • Add in-app feedback form
    • Monitor Chrome Web Store reviews
    • Send NPS surveys after usage

    Performance Optimization

    • Profile memory/CPU with Chrome DevTools
    • Lazy-load non-critical JS
    • Reduce bundle size with compression

    Scaling Strategies

    • Expand to Firefox, Edge, Safari
    • Add localization for new markets
    • Partner with SaaS platforms for integrations
